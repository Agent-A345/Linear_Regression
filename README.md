## 📈 Salary Prediction using Linear Regression

A machine learning project built using Python and Jupyter Notebook that applies Linear Regression to predict salaries based on years of experience. The project demonstrates data preprocessing, visualization, model training, prediction, and performance evaluation using a real-world salary dataset.

## 🚀 Features
* **Linear Regression Model** – Predict salaries using supervised machine learning.
* **Salary Prediction System** – Estimate salary based on years of experience.
* **Data Visualization** – Visualize relationships between experience and salary.
* **Data Preprocessing Pipeline** – Clean and prepare dataset for training.
* **Train-Test Splitting** – Separate training and testing datasets effectively.
* **Model Training & Evaluation** – Train regression model and evaluate accuracy.
* **Prediction Visualization** – Plot regression lines and prediction graphs.
* **Correlation Analysis** – Understand relationships between variables.
* **Real-World Dataset Application** – Learn regression using practical business data.
* **Beginner-Friendly ML Workflow** – Covers complete machine learning basics.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Linear_Regression.git
```
2. **Navigate to the project folder**:
```
cd Linear_Regression
```
3. **Install required libraries**
```
pip install pandas numpy matplotlib seaborn jupyter
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
linear_regression_project.ipynb
```
